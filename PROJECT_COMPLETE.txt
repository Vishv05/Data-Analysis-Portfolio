🚀 DEPLOYMENT OPTIONS

GITHUB PAGES (Free, Recommended for portfolios):
   1. Create GitHub account (if you don't have one)
   2. Create new repository: yourname/portfolio
   3. Push all files to main branch
   4. Go to Settings → Pages
   5. Select "Deploy from a branch"
   6. Choose "main"
   7. Wait 2-3 minutes
   8. Visit: yourname.github.io/portfolio

VERCEL (Free, Fast, Auto-deploys):
   1. Sign up at vercel.com
   2. Import GitHub repository
   3. Vercel auto-detects settings
   4. Click Deploy
   5. Auto-updates on every push
   6. Can add custom domain

NETLIFY (Free, Drag & Drop):
   1. Sign up at netlify.com
   2. Drag & drop the folder
   3. Done! Instant deploy
   4. Or: Connect GitHub for auto-deploys

❓ QUICK TROUBLESHOOTING

Q: Where do I make changes?
A: index.html for content, assets/js/main.js for projects, assets/css/custom.css 
   for styles

Q: Animations not working?
A: Check browser console (F12) for errors, ensure CDN links load, verify GSAP 
   and ScrollTrigger are available

Q: How do I add more projects?
A: Add project card HTML and corresponding projectData[4] object in main.js

Q: Form not sending emails?
A: Currently simulated. Integrate with Formspree, EmailJS, or your backend 
   for production

Q: How do I change colors?
A: Edit :root variables in assets/css/custom.css or change Tailwind classes

Q: Mobile looks wrong?
A: Test with DevTools (F12), check Tailwind responsive classes, adjust spacing
