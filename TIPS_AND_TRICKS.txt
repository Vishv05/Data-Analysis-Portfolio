FOR FORM SUBMISSION in production:
Currently the form is simulated. To make it actually send emails:

Option A: Formspree (Easiest)
   1. Go to formspree.io
   2. Sign up with your email
   3. Create form for this domain
   4. Add action="https://formspree.io/f/YOUR_ID" to form

Option B: EmailJS
   1. Sign up at emailjs.com
   2. Get API credentials
   3. Initialize in assets/js/main.js:
      emailjs.init("YOUR_PUBLIC_KEY");
   4. Update form submission code

Option C: Backend API
   1. Create backend endpoint
   2. Update form submission in assets/js/main.js
   3. Send data to your endpoint

COLORS:

Current scheme:
   Primary Blue:    #3b82f6
   Accent Cyan:     #22d3ee
   Dark:            #0f172a
   Text:            #ffffff

To change colors globally, edit assets/css/custom.css:

    :root {
        --color-blue-500: #YOUR_COLOR;
        --color-cyan-400: #YOUR_ACCENT;
    }

Or change Tailwind classes throughout HTML:
   Replace "blue-500" with your color (e.g., "indigo-500")
   Replace "cyan-400" with your color (e.g., "teal-400")

Color suggestions:
   Professional:  Blue/Purple/Teal
   Creative:      Purple/Magenta/Pink
   Nature:        Green/Teal/Yellow
   Tech:          Blue/Cyan/Purple

FONTS:

Current: Inter (default), Space Mono (for code)

To change in index.html:
    <link href="https://fonts.googleapis.com/css2?family=Poppins:wght@400;600;700&display=swap" rel="stylesheet">

Then update CSS:
    body { font-family: 'Poppins', sans-serif; }

ANIMATION SPEED:

GSAP duration: Edit assets/js/animations.js
   duration: 0.8  // seconds (slower = 1.2, faster = 0.4)

AOS duration: Edit index.html data attributes
   data-aos-duration="800"  // milliseconds

Typed.js speed: Edit assets/js/main.js
   typeSpeed: 50    // Lower = faster (try 30-50)
   backSpeed: 30    // Lower = faster delete

8. ADDING IMAGES

1. Add images to: assets/images/

2. Optimize before uploading:
   • Use WebP format with PNG fallback
   • Max width: 2000px
   • Max size: < 500KB per image
   • Tools: Tiny.png, Squoosh.app

3. Add to HTML:
   <img src="assets/images/your-image.png" alt="Description" class="w-full rounded-lg">

4. For hero background:
   Edit assets/css/custom.css in #hero section
   Add: background: url('assets/images/bg.png');

💡 FINAL PRO TIPS:

1. Keep your portfolio updated:
   - Add new projects as you complete them
   - Update skills as you grow
   - Refresh certifications section

2. Mobile first:
   - Always test on phone
   - Ensure touch targets (buttons) are 48px+
   - Test form on mobile

3. Performance:
   - Minify code before deployment
   - Optimize images
   - Lazy load images for speed

4. Credibility:
   - Link all projects to GitHub
   - Add actual project images
   - Use real metrics/results
   - Keep everything current

5. Stand out:
   - Focus on impact and results
   - Tell the story, not just features
   - Show problem → solution → impact
   - Use metrics that matter
