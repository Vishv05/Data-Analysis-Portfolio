🎯 GETTING STARTED (4 EASY STEPS):

1️⃣ OPEN IN BROWSER
   → Double-click index.html
   → Or use: Open with Live Server (VS Code)
   → Or run: python -m http.server 8000 (then visit localhost:8000)

2️⃣ CUSTOMIZE WITH YOUR INFO
   → Edit index.html with your name, email, phone, links
   → Update projects in assets/js/main.js (projectData object)
   → Add your skills percentages
   → Update stats counters

3️⃣ ADD YOUR CONTENT
   → Profile: Update hero section, about section
   → Skills: Modify skill bars and percentages
   → Projects: Add your 3+ best projects
   → Certifications: Add your achievements
   → Blog: Link to your actual blog posts
   → Contact: Your real email, phone, social links

4️⃣ DEPLOY ONLINE
   → GitHub Pages: Push to GitHub, enable Pages
   → Vercel: Connect GitHub repo, auto-deploy
   → Netlify: Drag & drop, or connect GitHub

🐛 TROUBLESHOOTING:

Issue: Animations not showing
→ Check browser console for errors
→ Ensure all CDN links are loading

Issue: Form not sending
→ Currently simulated, integrate with service for production
→ Options: Formspree, EmailJS, Firebase, backend API

Issue: Charts not displaying
→ Check canvas elements exist in HTML
→ Verify data is in correct format

Issue: Mobile responsive issues
→ Test with DevTools (F12)
→ Check Tailwind responsive classes
